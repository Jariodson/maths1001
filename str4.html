Основные понятия Видеоурок Таблицы производных ПЭФ Задачи Тест

Производные элементарных функций

Теорема 1. Для любого x ∈ R и любого натурального n ≥ 2 справедлива формула: (xn)` = nxn-1.
Теорема 2. Для любого x ∈ R, кроме x = 0, и любого нутурального числа n справедлива формула: (x-n)` = -nx-n-1.
Теорема 3. Пусть а > 0 и a ≠ 1, тогда для любого x ∈ R справедлива формула: (ax)` = axln(a). В частности, (ex)` = ex
Теорема 4. Пусть a > 0 и a ≠ 1, тогда для любого x > 0 справедлива формула: (loga(x))` = 1⁄xln(a).
Теорема 5. Для любого x ∈ R справедливы формулы: (sin(x))` = cos(x), (cos(x))` = -sin(x).
Теорема 6. Для любого x ≠ π⁄2 + πk, k ∈ Z, справедлива формула: (tg(x))` = 1⁄cos²(x). Для любого x ≠ πk, k ∈ Z, справедлива формула: (ctg(x))` = -1⁄sin²(x).
