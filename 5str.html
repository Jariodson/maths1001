Задачи

Задача 1. Для любого x ∈ R найдите производную функции
a)  y = x11; б)  y = x101; в)  y = x1001;
г)  y = 7x4 - 5x3 - x + 25; д)  y = -x4 + 8x2 + 2x - 19; е)  y = 12x5 - 20x3 - 30x²

Ответы

a)  (x11)` =  11x10 ; б)  (x101)` =  101x100; в)  (x1001)` =  1001x1000;
г)  (7x4 - 5x3 - x + 25)` =  28x³ - 15x² - 1; д)  (-x4 + 8x2 + 2x - 19)` =  -4x³ + 16x + 2;
е)  (12x5 - 20x3 - 30x²)` =  60x4 - 60² - 60x;
